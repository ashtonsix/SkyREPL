# collect-artifacts.ts - Collect Artifacts Node

## Input Type
{ runId: string, instanceId: string, patterns?: string[] }

## Output Type
{ artifactIds: string[], totalBytes: number, fileCount: number }

## Execute Steps
- Default patterns: ['**/*.pt', '**/*.safetensors', '**/*.log', 'checkpoints/**']
- Send CollectArtifactsMessage via SSE: { type: 'collect_artifacts', runId, patterns }
- Agent matches files in workdir, computes checksums
- Agent calls /v1/storage/check-blobs with checksums for dedup
- Control plane returns missing checksums with presigned upload URLs
- Agent uploads new blobs, reports completion via SSE event
- Control plane creates artifact objects linking run_id to blob object_ids
- Return list of artifact object IDs

## Compensation
- No compensation needed (artifacts are immutable once uploaded)
- Failed uploads will leave partial blobs in S3 (cleaned by orphan scanner)

## Key Calls
- sse.sendMessage(instanceId, { type: 'collect_artifacts', runId, patterns })
- sse.waitForEvent(instanceId, 'artifacts_collected', { runId })
- db.createObject() for each artifact
- db.addObjectTag(objectId, 'run_id', runId)
- storage.checkBlobsExist()
