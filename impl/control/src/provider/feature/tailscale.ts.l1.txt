# tailscale.ts - Tailscale Feature Provider (Ch8)

Cross-provider VPN capability via Tailscale. Attaches to existing instances.
See L2 for attach/detach logic, reconciliation, and bootstrap fragment generation.

## Implements
FeatureProvider<TailscaleConfig, TailscaleState>

## Core Methods
attach, detach, status

## Optional Methods
reconcile (detect orphans in tailnet vs DB), onInstanceHeartbeat, generateBootstrapFragment

## Capabilities
requiresBootstrap=true, supportsHotAttach=false, supportsDetach=true, crossProvider=true

## Reconciliation
- Query Tailscale API for machines with skyrepl:* tags
- Compare with DB feature_installations WHERE feature_type='tailscale'
- Report orphans (in API but not DB) and drift (in DB but offline/missing in API)

## Bootstrap Integration
Generates shell commands: curl installer, tailscale up with auth key, hostname, tags

## JIT Installation Note
JIT via SSH ProxyCommand handled by allocation system (not this feature provider).

## Materialization
Feature installations stored in DB. Tailscale machines tracked via API.
