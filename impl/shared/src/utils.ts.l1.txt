# utils.ts - Shared Utilities

## IDUtilities
- idToSlug(id: number): string - Convert to base-36
- slugToId(slug: string): number - Parse base-36 to number

## TimestampUtilities
- Timestamp.now(): TimestampMs
- Timestamp.fromDate(date): TimestampMs
- Timestamp.toDate(ts): Date
- Timestamp.addMs(ts, ms): TimestampMs
- Timestamp.isExpired(expiresAt): boolean

## DurationUtilities
- Duration constants: SECOND, MINUTE, HOUR, DAY
- Duration.seconds(n), minutes(n), hours(n), days(n)
- Duration.format(ms): string ("5s", "10m", "2.5h")
- Duration.parse(str): DurationMs

## Status Utilities
- parseStatus(status): { workflow, state }
- buildWorkflowState(workflow, phase): WorkflowState
- isActiveStatus(status): boolean
- isTerminalStatus(status): boolean
