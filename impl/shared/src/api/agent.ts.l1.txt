## CommandAcknowledgmentProtocol
- Commands include command_id (autoincrement integer)
- Agent queues pending_command_acks, sends in next heartbeat
- Control retries if no ack within 30s (max 3 attempts)
- Agent deduplicates via processed_command_ids (last 100, circular buffer)
