# resources.ts - API Request/Response Types

Standard API envelopes for requests, responses, pagination, errors.
See L2 for envelope schemas, error code mappings, resource CRUD types.

## RequestEnvelopes
ApiRequest, PaginationParams, SortParams, FilterParams, ListRequest

## ResponseEnvelopes
ApiResponse, ListResponse, ResponseMeta, PaginationMeta

## ErrorEnvelope
ApiError with code, message, category, details, retryAfterMs
ErrorCategory: validation | auth | not_found | conflict | rate_limit | provider | timeout | internal

## ErrorCodes
INVALID_INPUT, MISSING_REQUIRED_FIELD, INVALID_FORMAT, UNAUTHORIZED, FORBIDDEN, RESOURCE_NOT_FOUND, IDEMPOTENCY_CONFLICT, INVALID_STATE_TRANSITION, RATE_LIMITED, PROVIDER_ERROR, REQUEST_TIMEOUT, INTERNAL_ERROR

GetAllocationResponse, ListAllocationsRequest/Response, ExtendAllocationRequest/Response
GetWorkflowRequest/Response, ListWorkflowsRequest/Response, CancelWorkflowRequest/Response
GetManifestRequest/Response, ListManifestsRequest/Response
GetSnapshotResponse, ListSnapshotsRequest/Response, DeleteSnapshotResponse
GetArtifactResponse, DownloadArtifactResponse, ListRunArtifactsRequest/Response, UpdateArtifactRequest/Response
