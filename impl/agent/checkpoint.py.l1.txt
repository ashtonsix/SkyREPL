# checkpoint.py - Checkpoint Handling

Execute checkpoint_script on spot interruption or heartbeat panic with timeout enforcement.
See L2 for timeout budgets, artifact prioritization, failover URL support.

## Timeout Budgets
- Spot interruption: 90s checkpoint + 20s artifacts (2m total window)
- Heartbeat panic: 5m checkpoint + 4m artifacts (10m total window)

## Environment Variables
SKYREPL_CHECKPOINT_REASON (spot_termination | heartbeat_timeout), SKYREPL_RUN_ID, SKYREPL_WORKDIR

## Artifact Prioritization
- Spot (20s budget): Size ascending (maximize count)
- Panic (4m budget): Recency descending (prioritize recent work)

## Failover URLs
Each artifact has primary + optional failover URL (both 4h presigned, same S3 key).
On primary failure, retry with failover.

## Exports
execute_checkpoint, execute_spot_checkpoint, execute_panic_checkpoint, prioritize_artifacts, upload_artifacts, upload_to_url

## Integration
Runs after spot interruption detection OR heartbeat panic threshold, before artifact upload.

## Materialization
Subprocess execution, logs streamed to control plane.
