# artifact.ts - Artifact Materialization

## Material Backing
- Primary: objects table (type='artifact')
- External: S3/MinIO for large artifacts

## Key Tags (objects.tags_json)
- run_id, path, size_bytes, keep

## Storage Strategy
- ≤4KB: inline in objects.payload
- >4KB: S3, payload=null, use s3_key/s3_bucket

## Presigned URLs
- Generate for download regardless of storage location
- Same HTTP GET interface for both inline and S3

## Lifecycle
- Created when agent uploads
- keep=true exempts from auto-cleanup
- Deleted by cleanup workflow based on retention
