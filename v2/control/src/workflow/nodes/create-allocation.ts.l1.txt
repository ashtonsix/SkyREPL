# create-allocation.ts - Create Allocation Node

## Input Type
{ instanceId: string, runId: string, manifestId: string }

## Output Type
{ allocationId: string }

## Execute Steps
- Create allocation record in DB with:
  - instance_id: instanceId
  - run_id: runId
  - status: 'CLAIMED'
  - claimed_at: current timestamp
  - created_at: current timestamp
- Add allocation to manifest via ctx.emitResource()
- Return allocationId

## Compensation
- Delete allocation record: db.execute('DELETE FROM allocations WHERE id = ?', [allocationId])
- Note: Instance cleanup is handled separately by spawn-instance compensation

## Key Calls
- db.createAllocation({ instance_id, run_id, status: 'CLAIMED', claimed_at: now })
- ctx.emitResource('allocation', allocationId)
