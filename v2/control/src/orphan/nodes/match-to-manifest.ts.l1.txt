# match-to-manifest.ts - Match to Manifest Node

## Input Type
{ orphanedResources: OrphanedResource[] }

## Output Type
{ matched: Array<{ orphan: OrphanedResource, manifestId: string }>, unmatched: OrphanedResource[] }

## Execute Steps
- For each orphan in orphanedResources:
  - If orphan.inferredInfo.manifestIdPrefix exists:
    - Query db.getManifestByIdPrefix(manifestIdPrefix) to find matching manifest
    - If manifest found and status != 'sealed':
      - Add to matched array with { orphan, manifestId }
    - Else:
      - Add to unmatched array
  - Else (no manifest prefix or manifest prefix is 'none'):
    - Add to unmatched array
- Return { matched, unmatched }

## Compensation
- No compensation needed (read-only operation)

## Key Calls
- db.getManifestByIdPrefix(manifestIdPrefix)
- Context: ctx.emitMetadata('matched_count', matched.length)
- Context: ctx.emitMetadata('unmatched_count', unmatched.length)
