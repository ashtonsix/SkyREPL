# run.ts - Launch Run Command

## Input
- Profile from repl.toml (provider, spec, region)
- Command to execute (positional or --command)
- Workdir (default: current directory)

## Flags
- --timeout DURATION: Max run duration (default: 1h)
- --spot: Use spot instances (default: from profile)
- --hold DURATION: Hold allocation after completion (default: 5m)
- --snapshot: Create snapshot on completion
- --dry-run: Validate without submitting
- --no-stream: Don't stream progress (exit after submission)
- --env KEY=VALUE: Environment variables (repeatable)

## Flow
1. Resolve profile settings from repl.toml
2. Run preflight checks (unless --no-preflight)
3. Hash files for sync (POST /v1/blobs/check)
4. Upload missing blobs with progress
5. POST /v1/operations/launch-run

## Output
- Stream progress via ui/progress.ts (unless --no-stream)
- Receipt: run ID, SSH shortcut, artifact pull command
- Exit code: 0=success, 1=run failed, 2=CLI error
