# instance.ts - Instance Commands

## Subcommands
- list: GET /v1/instances
- terminate: POST /v1/operations/terminate-instance

## List Options
- --provider: Filter by provider
- --spec: Filter by instance type
- --status: Filter by workflow_state

## List Output
- Table: ID, provider, spec, IP, allocation count

## Terminate Options
- --force: Skip drain
- --drain-timeout: Wait for allocations (default 5m)
- Confirmation prompt with cost estimate
