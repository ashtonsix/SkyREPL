## #AGENT-04: Degraded mode                                        [Md] [DONE]
  Spec: Ch11 §11.5
  Depends: #FND-04
  ---
  Progressive degradation: healthy → degraded → panic.
  Degraded: control plane unreachable but run still active.
  Agent continues execution, buffers logs, retries control plane.

  Deliverables:
  - [x] State machine: healthy ↔ degraded → panic
  - [x] Log buffering during degraded mode (disk buffer, 10MB cap)
  - [x] Automatic recovery to healthy when control plane returns
  - [x] Configurable degradation thresholds
  - [x] Exponential backoff for heartbeat during degraded (10s→60s cap)
  - [x] Flush disk-buffered logs on recovery
