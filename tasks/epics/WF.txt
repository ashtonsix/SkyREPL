## #WF-03: Subworkflow contract                                   [Lg] [PENDING]
  Spec: Ch06 §6.4
  Depends: #WF-01
  ---

  Depth-limited (currently engine has depth check but no real support).

  Deliverables:
  - [ ] Subworkflow spawn from node context
  - [ ] Parent-child lifecycle binding (cancel parent → cancel children)
  - [ ] Failure propagation
  - [ ] Depth limit enforcement
  - [ ] Integration test: parent with 2 child subworkflows
